# Introdução
```
Este projeto foi desenvolvido no sétimo semestre na disciplina de Redes de Computadores A.
O seu objetivo foi criar um cliente e servidor utilizando os protocólos TCP/IP que consigam
listar os arquivos do servidor e realizar o envio ou recebimentos de arquivos do servidor.
```
# Começando a utilizá-lo
```
Este projeto deve ser utilizado em um sistema operacional Linux.
Primeiramente após realizar o download dos arquivos deve-se utilizar o comando make
na pasta onde os arquivos se encontram.
Serão gerados dois arquivos, um nomeado "client" e um nomeado "server", deve-se executar o arquivo 
server pelo terminal na pasta desejada e executar o arquivo client pelo terminal na pasta que 
receberá ou enviará os arquivos.
```
